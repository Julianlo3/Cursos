Inline

Se pueden modificar un parrafo, titulo o algo en la misma linea usando span.

Por ejemplo: esto sera modificado esto no sera modificado

DIVS

Los divs son bloques, ocupan su propio espacio y no pueden meterse dentro de otro como el span
Esto es muy útil para dividir secciones de la página y hacer cambios en CSS mediante los id y clases

Clases e id

Las clases son de mucha ayuda a la hora de de usar CSS, estas se aplican a cualquier etiqueta class="..." y en CSS con .(nombreClase)
Se puede aplicar a una etiqueta en concreto y una clase por ejemplo: .tablaGeneral th --> esto aplicará el CSS solo a los th con clase tablaGeneral
Los id son similares a las clases pero estas son de elementos únicos se usa con id='..' y en CSS #(nombre), tales como titulo general, con esto uno puede ir navengando en la misma página usando href
