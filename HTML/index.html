Elementos esenciales para la creación de una página básica
Elemento Descripción Etiquetas que se relacionan Otra información
h1,h2,h3,...,hn Se usa para los titulos, y va en cascada, h1 es más grande que h2 y así br
(Separador de linea)
pre
( Es como p pero este deja tal cual se escribe con espacios) 
            b ( Negrita)
            strong (Es como b pero con prioridad )
            i (Texto en cursiva ulala)
            em (Enfatizar es como i pero con prioridad)
            small (Texto pequeño)
            mark (Texto marcado)
            del (Texto tachado)
            ins (Texto con linea abajo)
            sub y sup (Textocon subindice y texto con superindice)
            code (Texto en formato de código)
p Se usa para los parrafos
hr Pone una linea horizontal a modo de separador
css básico Estilo en cascada que se usa para darle diseño a la página 
          text-alight (Permite cambiar la posición del texto)
          color (a color al texto)
          background-color (Da color al fondo)
          font-family (Permite cambiar el tipo de letra)
          font-size (Permite cambiar el tamaño de letra)
          border (Permite bordear)
          padding y margin ( Permite colocar un espacio al rededor, una margen)
Es recomendable usar un archivo aparte .css que se enlaza con link href y rel="stylesheet"
a - anclaje - Hipervinculos sencillos Los hipervinculos se pueden manejar tanto locales como por direcciones de internet Se usa con a y href Abrir esta página Se puede usar el atributo target="_blank" para abrir en una nueva ventana.
Además se puede mover por la página mediante los id de CSS usando href="#.."
img - src- Imagenes Para las imagenes se usa img y src que es la fuente de donde proviene la imagen, se puede tanto local como en internet 
            alt ( texto alternativo cuando la imagen no carga)
            title ( texto cuando se pasa el mouse por encima)
            with ( tamaño de la imagen)
