Formularios en HTML

Los formularios son información básica de un usuario para, por ejemplo, un login

Etiquetas básicas para los formularios HTML
Etiqueta Descripción Etiquetas/atributos relacionadas Otra información
form creación como tal de la tabla, aunque no se visualiza
  • input
  • Method
 El method es como se envia la información, por default es GET que envia directo en la URL.
Si se maneja información sensible usar el método POST HTTP
input id = "..." (el id no es obligatorio pero es recomendable) Especifica que va hacer un espacio para ingresar datos
  • value=".." - pone valores directamente en los inputs
  • name="..." - se nombra como tal el input para enviar los datos
  • placeholder="..." - texto que viene dentro de los inputs. Esta no se envia
  • required="true" - especificar que un campo es obligatorio
 Puede ser de tipo:
  • button
  • checkbox
  • date
  • datetime
  • email
  • file
    • ...
labbel Un labbel es una etiqueta escrita que permite saber que información se va a llenar.
  • for="..." - relaciona un labbel con el input

Methot GET y POST HTTP

El atributo methot hace referencia a como se envia la información. Si se usa GET (default) la información se enviará directamente en el URL de la página, en cambio con POST se envia usando HTTPS
