Curso de git-principios

Conceptos básicos

Git es un control de versiones que permite hacer copias de elementos generalmente de código, para guardar un historial. Inventor: Linus Torval

Para el manejo de git se hace necesario el control por consola. En git se usa el Git Bash

Comandos bácicos de consola
Comando Descripción Otra información
ls Lista los elementos de la carpeta actual
cd Cambio de directorio (posición) Puede ser posición absoluta (dirección completa)
Puede ser relativa
  • Posición actual ( ./ )
  • Pósición atrás ( cd ..)
  • Cambio posición ( cd directorio/)
pwd Mostrar toda la ruta del directorio actual
touch Creación de cualquier archivo: touch "ejemplo.txt"
mkdir Creación de carpetas Cuando se crea no se cambia automaticamente a la carpeta creada. Toca usar cd
Comandos básicos de git
Comando Descripción Otra información
  • git config --global user.name "..."
  • git config --global user.email "..."
 Estos dos comandos son los primeros pasos para iniciar con git. Siempre se necesita un usuario y un correo Los datos quedan guardados localmente y se pueden ver en .gitconfig
init El comando más importante, indica que el directorio actual será un repositorio Se crea un fichero oculto llamado .git en el directorio actual

Cuando se crea por primera vez, se tiene la rama principal master o main
git status Sirve para ver el estado actual, ver archivos si han cambiado
